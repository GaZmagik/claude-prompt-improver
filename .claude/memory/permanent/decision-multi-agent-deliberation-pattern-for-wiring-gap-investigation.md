When discovering integration gaps in modular systems: (1) Frame problem with Architect perspective (2) Challenge with Devils-Advocate (3) Add test-quality perspective to identify coverage gaps (4) Security-review to check for risks (5) Pragmatist to define minimum viable fix. Promotes structured root-cause analysis and prevents overlooking interdependencies.
